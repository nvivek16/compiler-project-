Content based Ad Classification
 
Requirements - 
OS - Linux, OSX


Dependencies -
Python, PHP, Bash, JavaScript, CSS, HTML, Beautiful Soup(Python module), SVM multiclass binary files


To execute -
$ sh run_command


1)Put the folder compiler inside apache server
2)If you add any extra test data to test data folder, add it the mail folder also
3)Then open your web browser and type - localhost/compiler/display1.php


Files - 
display1.php - Display ads
maillist.php - Display mails
classify.py - Format data for training
test.py - Format data for testing
news.py - Web scraper
