Content Based Ad Classification


Project Description

Our project is to display advertisements based on the mails received. We have taken three categories into account namely - Sports, Education, and Health. Using a training set, we train the system to understand how a mail of a particular category will look. Our system uses Support Vector Machines Multiclass Algorithm. Next, we tried to test our system using a set of mails. SVM classifies each mail to a category. We then display advertisements of the category which have maximum mails.



Web Scraper

We used a web scrapper written in Python to automatically scrape various data from the internet which was used as the training set for the system. By using scrapper we were able to train the system with large amounts of data.



SVM Multiclass

Multiclass SVM aims to assign labels to instances by using support vector machines, where the labels are drawn from a finite set of several elements. The dominant approach for doing so is to reduce the single multiclass problem into multiple binary classification problems.



Front End

The advertisements are displayed using a web application written in PHP. The Python backend writes the results to a file which is read by the PHP application. The web app then walks through the advertisements of the selected category and sends it as a JSON file object to the webapp in the form of an AJAX request.
